How it works

Stardew saves are tied to Wendy's account. A little swap tool updates the save file so a different person becomes the host for that session. Then everyone grabs the updated file from Discord before playing!

🎀 host Runs the swap (or asks Wendy), zips the save, uploads to Discord, loads the farm and invites everyone
🌸 everyone Downloads the zip from our saves channel on Discord, extracts it, puts it in the right folder, then joins the host

The host can do the swap themselves! You'll need the two tool files (they'll be included in the .zip).

1

Download the latest save .zip from our saves channel on Discord and install it first — see Installing the Save File below.

2

Double-click Swap-StardewHost.bat. A window will pop up! 🌸

3

It shows a list of saves. Type the number next to catnobboptob and press Enter.

4

It shows everyone's names. Type the number next to your name and press Enter.

5

When the host swap completes, you can close the batch window. ✅

6

Zip the updated save folder and upload it to Discord in our saves channel so everyone can grab it!

🗜️ How to zip the save folder
➊Press Win + R, type %AppData%\StardewValley\Saves, press Enter
➋Right-click the catnobboptob_438842613 folder
➌Windows 11: Compress to ZIP file  ·  Windows 10: Send to → Compressed folder
➍Drag the new .zip into our saves channel on Discord and send! 💌
📁

Installing the save file — everyone does this!

1

Download the .zip file from our saves channel on Discord.

2

Right-click the .zip → Extract All → click Extract. A folder will appear! ✨

3

Open your Stardew saves folder — press Win + R, paste this, press Enter:

%AppData%\StardewValley\Saves
4

Drag the extracted catnobboptob_438842613 folder into the Saves window. If asked to replace the old one, click Yes.

5

All done — you're ready to play! 🌸

Loading the game

👑
If you're the host: Co-op → Host → select catnobboptob farm → start! Then send everyone the invite code from the in-game pause menu.
