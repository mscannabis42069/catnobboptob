How it works

Stardew saves are tied to Wendy's account. A little swap tool updates the save file so a different person becomes the host for that session. Then everyone grabs the updated file from Discord before playing!

🎀 host Runs the swap (or asks Wendy), zips the save, uploads to Discord, loads the farm and invites everyone
🌸 everyone Downloads the zip from Discord, extracts it, puts it in the right folder, then joins the host

The host can do the swap themselves! You'll need the two tool files (they're in the saves channel on our discord server) and Python installed — see Installing Python at the bottom.

➊Press Win + R, type %AppData%\StardewValley\Saves, press Enter
➋Right-click the catnobboptob_438842613 folder
➌Windows 11: Compress to ZIP file  ·  Windows 10: Send to → Compressed folder
➍Drag the new .zip into Discord and send! 💌

Installing the save file — everyone does this!

1

Download the .zip file from Discord.

2

Right-click the .zip → Extract All → click Extract. A folder will appear! ✨

3

Open your Stardew saves folder — press Win + R, paste this, press Enter:

%AppData%\StardewValley\Saves
4

Drag the extracted catnobboptob_438842613 folder into the Saves window. If asked to replace the old one, click Yes.

5

All done — you're ready to play! 🌸

💡 If you don't see a Saves folder, just launch Stardew once and it'll create one automatically!

Installing Python — only if you're running the swap tool

💜
Who needs this? Only the person doing the swap when Wendy is asleep. If Wendy always does it, skip this section entirely!

The easiest way is through the Microsoft Store — no confusing websites! 🌸

1

Press the Windows key, type Microsoft Store, and open it.

2

Search for Python Install Manager in the search bar at the top.

3

Click the result made by Python Software Foundation. It is free — if it asks for money, it's the wrong one! 🚨

4

Click Get, wait for it to finish, then click Open.

5

A window pops up with setup questions. For each one, type y and press Enter. It's safe to say yes to everything!

6

When it shows a big list of commands, type n and press Enter, then close the window.

7

Restart your computer! Python won't work properly until you do. 💜

✨ You only ever need to do this once! After restarting, run_host_swap.bat will work whenever you need it.
